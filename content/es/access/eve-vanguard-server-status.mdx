<Section icon={Swords} title="Solución de problemas de conexión">

Si la página de estado del servidor indica que todos los sistemas están operativos, pero aún no puedes conectarte, es probable que el cuello de botella se encuentre en tu red local o en la configuración del launcher. Los síntomas comunes de problemas de conectividad local incluyen pantallas de carga infinitas, fallos de autenticación o ser devuelto al menú principal inmediatamente después de que comience el emparejamiento.

Sigue estos pasos estructurados para diagnosticar y reparar problemas de red comunes que te impiden unirte a la batalla.

<Steps>
  <Step num="1" title="Verificar el estado de la red">
    Consulta la página de estado oficial para confirmar que el clúster Tranquility y la API Quasar están totalmente operativos antes de cambiar la configuración local.
  </Step>
  <Step num="2" title="Reiniciar el EVE Launcher">
    Cierra el launcher completamente desde la bandeja del sistema y reinícialo para forzar una comprobación de actualizaciones del cliente y refrescar tu sesión de inicio de sesión.
  </Step>
  <Step num="3" title="Vaciar la caché de DNS local">
    Abre el símbolo del sistema como administrador y ejecuta el comando para borrar las rutas de enrutamiento desactualizadas hacia los servidores de emparejamiento de AWS.
  </Step>
  <Step num="4" title="Verificar la integridad de los archivos del juego">
    Usa la configuración del launcher para escanear y verificar los archivos del juego instalados, reparando cualquier activo dañado que cause cierres inesperados durante la carga.
  </Step>
</Steps>

Si encuentras errores específicos durante estos pasos, consulta la tabla a continuación para identificar la causa raíz y aplicar la resolución adecuada.

| Código de error | Causa potencial | Solución recomendada |
| :--- | :--- | :--- |
| Tiempo de espera agotado | Pérdida de paquetes local o problemas de ruta | Cambia a una conexión por cable y reinicia el router |
| Discordancia de versión | Archivos del cliente desactualizados en tu disco | Fuerza una comprobación manual de actualizaciones en el launcher |
| Fallo de autenticación | Token de seguridad de la cuenta caducado | Cierra la sesión en el launcher por completo y vuelve a entrar |

<Callout type="tip" title="Consejo Pro">
Ejecutar el EVE Launcher con privilegios de administrador evita que Windows Defender bloquee actualizaciones esenciales en segundo plano.
</Callout>

</Section>
