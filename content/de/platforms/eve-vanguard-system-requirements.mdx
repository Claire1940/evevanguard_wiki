- **EVE Vanguard Systemanforderungen**: Erfordert ein **64-Bit Windows 10/11** Betriebssystem und **50 GB SSD** Speicherplatz.

<Section icon={Laptop} title="EVE Vanguard Systemanforderungen: PC-Spezifikationen">

Bevor Sie als unsterblicher Warclone auf die hochriskanten planetaren Schlachtfelder von New Eden verlegt werden, müssen Sie sicherstellen, dass Ihr Gaming-Rig optimal vorbereitet ist. EVE Vanguard basiert auf einer modernen Engine, die düstere, immersive Sci-Fi-Visuals liefert, was bedeutet, dass die Hardware-Anforderungen im Vergleich zu herkömmlichen taktischen Shootern relativ hoch sind.

Das Spiel benötigt zwingend ein Solid-State-Drive (SSD), um das Asset-Streaming während intensiver PvPvE-Extraktionsmissionen nahtlos zu bewältigen. Unten finden Sie die offizielle Aufschlüsselung der Hardware-Spezifikationen, zusammengestellt von der [EVE Vanguard Steam-Seite](https://store.steampowered.com/app/1872800/EVE_Vanguard/) und Entwickler-Updates.

| Hardware-Komponente | Mindestanforderung | Empfohlene Spezifikation |
| :--- | :--- | :--- |
| **Betriebssystem** | Windows 10 64-Bit (21H2/19044) | Windows 10/11 64-Bit (Aktuellste Version) |
| **Prozessor (CPU)** | Intel i5-8400 / AMD Ryzen 5 1600X | Intel i5-11600K / AMD Ryzen 5 5600X |
| **Grafikkarte (GPU)** | NVIDIA RTX 2060 / AMD RX 5700 XT | NVIDIA RTX 3080 / AMD RX 6900 XT |
| **Videospeicher (VRAM)** | 6 GB | 8 GB oder höher |
| **Arbeitsspeicher (RAM)** | 16 GB DDR4 | 16 GB DDR4 |
| **Speicherplatz** | 50 GB freier Platz (SSD erforderlich) | 50 GB freier Platz (NVMe SSD empfohlen) |
| **DirectX-Version** | DirectX 12 | DirectX 12.1 |
| **Netzwerkverbindung** | Breitband-Internet | Breitband-Internet |

<Callout type="warning" title="SSD ist zwingend erforderlich">
Die Installation von EVE Vanguard auf einer herkömmlichen Festplatte (HDD) wird dringend nicht empfohlen. Dies führt zu schweren Textur-Pop-ins, verlängerten Ladezeiten und potenziellen Rucklern während Feuergefechten. Verwenden Sie immer eine SSD oder NVMe-Laufwerk.
</Callout>

</Section>

<Section icon={Shield} title="Anti-Cheat und Software-Anforderungen">

Um eine faire Spielumgebung in einem High-Stakes Extraction-Shooter zu gewährleisten, bei dem der Tod den Verlust hart erarbeiteter Beute bedeutet, implementiert EVE Vanguard robuste Sicherheitsmaßnahmen.

Das Spiel nutzt **Easy Anti-Cheat (EAC)**, ein Anti-Cheat-System auf Kernelebene. Diese Software wird zusammen mit dem Game-Launcher initialisiert, um Speicher-Modifier von Drittanbietern und unbefugte Code-Injektionen zu erkennen.

| Software-Element | Anforderung & Auswirkung | Erforderliche Aktion |
| :--- | :--- | :--- |
| **Anti-Cheat-System** | Easy Anti-Cheat (EAC) | Automatische Installation über Steam/EVE Launcher |
| **BS-Architektur** | Strikte 64-Bit-Anforderung | Stellen Sie sicher, dass Ihr Betriebssystem vollständig aktualisiert ist |
| **Netzwerksicherheit** | Permanente Online-Verbindung | Offener NAT-Typ für Matchmaking empfohlen |
| **Deinstallation** | Manuelle Bereinigung eventuell nötig | EAC kann über die Windows-Systemsteuerung entfernt werden |

<Steps>
  <Step num="1" title="Spiel starten">
    Starten Sie EVE Vanguard über Steam oder den EVE Launcher während der aktiven Playtest-Fenster.
  </Step>
  <Step num="2" title="EAC-Initialisierung">
    Lassen Sie Easy Anti-Cheat laden und Ihre Systemdateien verifizieren, bevor das Hauptmenü erscheint.
  </Step>
  <Step num="3" title="Netzwerk-Handshake">
    Stellen Sie sicher, dass Ihre Firewall die ausgehenden Verbindungen von Vanguard zu den Matchmaking-Servern nicht blockiert.
  </Step>
</Steps>

<Callout type="warning" title="EAC-Kompatibilitätshinweis">
Einige Hintergrundanwendungen, Overlay-Software oder benutzerdefinierte Systemkerne können Kompatibilitätswarnungen mit Easy Anti-Cheat auslösen. Wenn das Spiel nicht startet, versuchen Sie, nicht essenzielle Hintergrundprozesse zu deaktivieren.
</Callout>

</Section>

<Section icon={Swords} title="Steam Deck und Linux Kompatibilität">

Für Spieler, die mobiles Gaming bevorzugen oder Linux-basierte Betriebssysteme nutzen, erfordert die Kompatibilität von EVE Vanguard besondere Aufmerksamkeit. Da das Spiel auf kompetitiven PvPvE-Extraktionselementen basiert, wirken sich die Sicherheitsebene direkt auf Nicht-Windows-Plattformen aus.

| Plattform / BS | Kompatibilitätsstatus | Hinweise & Workarounds |
| :--- | :--- | :--- |
| **Steam Deck** | Nicht unterstützt / Experimentell | Easy Anti-Cheat (EAC) unterstützt derzeit noch keine Proton-Runtimes für Vanguard offiziell. |
| **Linux (Proton)** | Eingeschränkt | Erfordert manuelle Konfiguration der EAC-Runtimes; Performance ist nicht garantiert. |
| **Windows-Handhelds** | Unterstützt | Handhelds mit nativem Windows (z. B. ROG Ally, Legion Go) können das Spiel ausführen, wenn die Mindest-GPU-Anforderungen erfüllt sind. |

<Callout type="info" title="Zukünftige Kompatibilitäts-Updates">
Die Entwickler sammeln während der Alpha-Playtests aktiv Feedback. Die offizielle Proton- und Steam-Deck-Unterstützung könnte näher am geplanten Launch-Fenster im 3. Quartal 2026 aktualisiert werden. Schauen Sie im [EVE Vanguard Discord](https://discord.gg/evevanguard) nach Community-basierten Linux-Workarounds.
</Callout>

</Section>
