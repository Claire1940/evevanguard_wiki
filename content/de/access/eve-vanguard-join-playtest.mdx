<Section icon={Shield} title="PC-Systemanforderungen & Leistung">

Bevor Sie versuchen, dem **eve vanguard join playtest** beizutreten, müssen Sie sicherstellen, dass Ihre Hardware die hochauflösenden Unreal Engine 5-Umgebungen bewältigen kann. EVE Vanguard ist ein anspruchsvoller Titel, der fortschrittliche Beleuchtung und Physik nutzt, um die rauen Bedingungen des planetaren Kampfes zu simulieren.

<Callout type="warning" title="Anti-Cheat-Anforderung">
EVE Vanguard verwendet **Easy Anti-Cheat (EAC)** auf Kernel-Ebene. Stellen Sie sicher, dass Ihr System kompatibel ist und keine widersprüchliche Software ausgeführt wird, da das Spiel andernfalls nicht startet.
</Callout>

### Spezifikationstabelle für Hardware

| Komponente | Mindestanforderungen | Empfohlene Spezifikationen |
| :--- | :--- | :--- |
| **Betriebssystem** | Windows 10 64-Bit (21H2) | Windows 11 64-Bit (Aktuellste Version) |
| **Prozessor** | Intel i5-8400 / AMD Ryzen 5 1600X | Intel i5-11600K / AMD Ryzen 5 5600X |
| **Grafikkarte** | NVIDIA RTX 2060 / AMD RX 5700 XT | NVIDIA RTX 3080 / AMD RX 6900 XT |
| **Arbeitsspeicher** | 16 GB DDR4 | 16 GB DDR4/DDR5 |
| **Speicherplatz** | 50 GB SSD | 50 GB NVMe SSD |
| **DirectX** | Version 12 | Version 12.1 |

Die Leistung hängt stark von der SSD-Geschwindigkeit ab. Wenn Sie das Spiel auf einer herkömmlichen HDD installieren, führt dies zu erheblichen Rucklern und langen Ladezeiten, was in einem Extraction-Shooter, in dem jede Sekunde zählt, fatal sein kann. Wenn Sie mit den Mindestspezifikationen 60 FPS bei 1080p anstreben, wird dringend empfohlen, **FSR** oder **DLSS** im Modus „Qualität“ oder „Ausgewogen“ zu aktivieren.

<Section icon={Target} title="Leitfaden für Anfänger: Ihr erster Einsatz">

Sobald Sie erfolgreich dem **eve vanguard join playtest** beigetreten sind, kann Ihr erster Einsatz überwältigend sein. Sie beginnen auf der Warbarge, Ihrer orbitalen Operationsbasis, wo Sie Ihre Ausrüstung vorbereiten, bevor Sie auf die Planetenoberfläche abgesetzt werden. Das Ziel ist theoretisch einfach, in der Ausführung jedoch komplex: Schließen Sie Ihren Vertrag ab und entkommen Sie lebend.

<Steps>
  <Step num="1" title="Einen Vertrag auswählen">
    Wählen Sie vor dem Einsatz einen Vertrag aus dem verfügbaren Pool aus. Diese reichen von einfacher Ressourcensammlung bis hin zur Eliminierung hochrangiger Offiziere von Mordu's Legion.
  </Step>
  <Step num="2" title="Absetzen und Plündern">
    Sie kommen in einer Landekapsel an. Suchen Sie sofort Deckung und scannen Sie das Gebiet nach „Absturzstellen“ oder „Vorposten“. Diese enthalten die hochkarätige Beute, die für Upgrades benötigt wird.
  </Step>
  <Step num="3" title="Kämpfen oder Ausweichen">
    Nutzen Sie Ihren Scanner, um Wärmesignaturen zu orten. Wenn Sie einen Trupp gegnerischer Warclones sehen, entscheiden Sie, ob Sie die nötige Feuerkraft für einen Sieg haben. Denken Sie daran: Wer stirbt, verliert alle nicht extrahierte Beute.
  </Step>
  <Step num="4" title="Den Command Block sichern">
    In Operation Avalon bietet das Vorstoßen zu den zentralen Kartenzielen die besten Belohnungen, zieht aber auch die gefährlichsten Feinde an.
  </Step>
  <Step num="5" title="Sicher extrahieren">
    Achten Sie auf der Karte auf aktive Extraktionssymbole. Begeben Sie sich in die Zone, rufen Sie den Transport und verteidigen Sie das Gebiet, bis der Timer Null erreicht.
  </Step>
</Steps>

<Callout type="tip" title="Die „Goldene Regel“ der Extraktion">
Es ist immer besser, frühzeitig mit einem kleinen Gewinn zu extrahieren, als für „noch eine Kiste“ zu bleiben und alles zu verlieren. Gier ist die Hauptursache für das Scheitern in EVE Vanguard.
</Callout>

</Section>

<Callout type="info" title="Modulare Chipsätze">
Sie können Ihre Waffen mithilfe von **Chipsätzen** anpassen. Diese ermöglichen es Ihnen, mitten im Einsatz die Schadensart zu wechseln (z. B. von Kinetisch auf Energie) oder die Feuermodi zu ändern (z. B. von Feuerstoß auf Vollautomatisch), sofern Sie die erforderlichen Komponenten finden.
</Callout>

### Empfohlene Loadouts

- **Der Aufklärer (The Scout)**: Laser DMR + SMG. Konzentriert sich auf hohe Mobilität und Aufklärung auf große Distanz. Ideal für Solospieler, die direkte Konfrontationen lieber vermeiden.
- **Der Vollstrecker (The Enforcer)**: Shrapnel Cannon + Beam Rifle. Konzipiert für den truppbasierten Kampf an vorderster Front. Erfordert intensives Ressourcenmanagement, glänzt aber beim Säubern von Vorposten.
- **Der Berger (The Salvager)**: SMG + Nova Blade. Optimiert für Schnelligkeit und Tarnung. Konzentriert sich auf das Plündern der Außenbezirke und die Extraktion, bevor die Hauptkämpfe beginnen.
