<Section icon={Shield} title="PC System Requirements & Performance">

Before you attempt to **eve vanguard join playtest**, you must ensure your hardware can handle the high-fidelity Unreal Engine 5 environments. EVE Vanguard is a demanding title, utilizing advanced lighting and physics to simulate the harsh conditions of planetary combat.

<Callout type="warning" title="Anti-Cheat Requirement">
EVE Vanguard utilizes **Easy Anti-Cheat (EAC)** at the kernel level. Ensure your system is compatible and that no conflicting software is running, or the game will fail to launch.
</Callout>

### Hardware Specification Table

| Component | Minimum Requirements | Recommended Specs |
| :--- | :--- | :--- |
| **OS** | Windows 10 64-bit (21H2) | Windows 11 64-bit (Latest) |
| **CPU** | Intel i5-8400 / AMD Ryzen 5 1600X | Intel i5-11600K / AMD Ryzen 5 5600X |
| **GPU** | NVIDIA RTX 2060 / AMD RX 5700 XT | NVIDIA RTX 3080 / AMD RX 6900 XT |
| **RAM** | 16 GB DDR4 | 16 GB DDR4/DDR5 |
| **Storage** | 50 GB SSD | 50 GB NVMe SSD |
| **DirectX** | Version 12 | Version 12.1 |

Performance is heavily dependent on SSD speed. Installing the game on a traditional HDD will result in significant stuttering and long load times, which can be fatal in an extraction shooter where every second counts. If you are targeting 60 FPS at 1080p on minimum specs, it is highly recommended to enable **FSR** or **DLSS** in the "Quality" or "Balanced" mode.
