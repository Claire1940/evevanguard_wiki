<Section icon={Laptop} title="EVE Vanguard System Requirements: PC Specifications">

Before deploying as an immortal warclone into the high-stakes planetary battlefields of New Eden, you must ensure your gaming rig is fully prepared. EVE Vanguard is built on a modern engine designed to deliver dark, immersive sci-fi visuals, meaning hardware demands are relatively high compared to traditional tactical shooters. 

The game requires a solid-state drive (SSD) to handle asset streaming seamlessly during intense PvPvE extraction missions. Below is the official hardware specification breakdown compiled from the [EVE Vanguard Steam Page](https://store.steampowered.com/app/1872800/EVE_Vanguard/) and developer updates.

| Hardware Component | Minimum Requirement | Recommended Specification |
| :--- | :--- | :--- |
| **Operating System** | Windows 10 64-bit (21H2/19044) | Windows 10/11 64-bit (Latest) |
| **Processor (CPU)** | Intel i5-8400 / AMD Ryzen 5 1600X | Intel i5-11600K / AMD Ryzen 5 5600X |
| **Graphics Card (GPU)** | NVIDIA RTX 2060 / AMD RX 5700 XT | NVIDIA RTX 3080 / AMD RX 6900 XT |
| **Video Memory (VRAM)** | 6 GB | 8 GB or higher |
| **System Memory (RAM)** | 16 GB DDR4 | 16 GB DDR4 |
| **Storage Space** | 50 GB free space (SSD required) | 50 GB free space (NVMe SSD recommended) |
| **DirectX Version** | DirectX 12 | DirectX 12.1 |
| **Network Connection** | Broadband Internet | Broadband Internet |

<Callout type="warning" title="SSD is Mandatory">
Installing EVE Vanguard on a traditional Hard Disk Drive (HDD) is highly discouraged. Doing so will lead to severe texture pop-in, extended loading times, and potential stuttering during firefights. Always use an SSD or NVMe drive.
</Callout>

<Section icon={Shield} title="Anti-Cheat and Software Requirements">

To ensure a fair play environment in a high-stakes extraction shooter where dying means losing your hard-earned loot, EVE Vanguard implements robust security measures. 

The game utilizes **Easy Anti-Cheat (EAC)**, a kernel-level anti-cheat system. This software initializes alongside the game launcher to detect third-party memory modifiers and unauthorized code injections.

| Software Element | Requirement & Impact | Action Required |
| :--- | :--- | :--- |
| **Anti-Cheat System** | Easy Anti-Cheat (EAC) | Automated installation via Steam/EVE Launcher |
| **OS Architecture** | Strict 64-bit requirement | Ensure your operating system is fully updated |
| **Network Security** | Persistent online connection | Open NAT type recommended for matchmaking |
| **Uninstallation** | Manual cleanup may be required | EAC can be removed via Windows Control Panel |

<Steps>
  <Step num="1" title="Launch the Game">
    Launch EVE Vanguard via Steam or the EVE Launcher during active playtest windows.
  </Step>
  <Step num="2" title="EAC Initialization">
    Allow Easy Anti-Cheat to load and verify your system files before the main menu screen appears.
  </Step>
  <Step num="3" title="Network Handshake">
    Ensure your firewall does not block Vanguard's outbound connections to the matchmaking servers.
  </Step>
</Steps>

<Callout type="warning" title="EAC Compatibility Notice">
Some background applications, overlay software, or custom system kernels may trigger compatibility warnings with Easy Anti-Cheat. If the game fails to launch, try disabling non-essential background processes.
</Callout>

</Section>

<Section icon={Swords} title="Steam Deck and Linux Compatibility">

For players who prefer mobile gaming or run Linux-based operating systems, EVE Vanguard's compatibility requires special attention. Because the game is built around competitive PvPvE extraction elements, its security layers directly impact non-Windows platforms.

| Platform / OS | Compatibility Status | Notes & Workarounds |
| :--- | :--- | :--- |
| **Steam Deck** | Unsupported / Experimental | Easy Anti-Cheat (EAC) does not officially support Proton runtimes for Vanguard yet. |
| **Linux (Proton)** | Limited | Requires manual configuration of EAC runtimes; performance is not guaranteed. |
| **Windows Handhelds** | Supported | Handhelds running native Windows (e.g., ROG Ally, Legion Go) can run the game if minimum GPU requirements are met. |

<Callout type="info" title="Future Compatibility Updates">
The developers are actively gathering feedback during the Alpha playtests. Official Proton and Steam Deck compatibility may be updated closer to the planned Q3 2026 launch window. Check the [EVE Vanguard Discord](https://discord.gg/evevanguard) for community-driven Linux workarounds.
</Callout>
