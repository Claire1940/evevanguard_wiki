Entering a planetary battlefield in EVE Vanguard is a high-pressure, high-risk endeavor. Unlike traditional arena shooters, this is an extraction-adventure FPS where survival is your only guarantee of progression. If you die before reaching an extraction zone, you lose all the gear, resources, and tech salvaged during that session.

### The Core Gameplay Cycle

To establish yourself as a successful warclone, you must master the fundamental loop of deployment, resource acquisition, and tactical retreat.

<Steps>

  <Step num="3" title="Raid and Salvage">

<Section icon={Shield} title="EVE Vanguard PC System Requirements">

Before requesting access to the **eve vanguard steam** playtest, verify that your computer meets the official hardware requirements. Because the game features highly detailed environments and complex physics, a solid-state drive (SSD) is mandatory.

### System Requirements Table

| Hardware Component | Minimum Requirement | Recommended Specification |
| :--- | :--- | :--- |
| **Operating System** | Windows 10 (64-bit, 21H2) | Windows 10 / 11 (64-bit latest) |
| **Processor (CPU)** | Intel i5-8400 / AMD Ryzen 5 1600X | Intel i5-11600K / AMD Ryzen 5 5600X |
| **Graphics Card (GPU)** | NVIDIA RTX 2060 / AMD RX 5700 XT | NVIDIA RTX 3080 / AMD RX 6900 XT |
| **System Memory (RAM)** | 16 GB DDR4 | 16 GB DDR4 |
| **Video Memory (VRAM)** | 6 GB | 8 GB+ |
| **Storage Space** | 50 GB available SSD space | 50 GB available NVMe SSD space |
| **DirectX Version** | DX 12 / DX 12.1 | DX 12.1 |
| **Anti-Cheat System** | Kernel-level Easy Anti-Cheat | Kernel-level Easy Anti-Cheat |

<Callout type="warning" title="Anti-Cheat Notice">
